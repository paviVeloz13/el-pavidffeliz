# El PaviDFeliz v0.1.0 Prototype

Prototype release for **macOS Apple Silicon (arm64)**.

## Known issue

The currently uploaded DMG is **not Developer ID signed or notarized**.

- macOS Gatekeeper may report the app as **damaged**
- a signed and notarized rebuild is pending
- do not use the current GitHub asset for normal installation

This warning will be removed after the rebuilt artifact replaces the current one.

## Included asset

- `El PaviDFeliz-0.1.0-arm64.dmg`

## What users can do in this build

- Convert PDF pages to PNG or JPEG
- Convert JPEG, PNG, and WEBP images
- Build a PDF from multiple images
- Merge, split, reorder, and delete PDF pages
- Compress PDFs and images
- Add text, highlight, and strikethrough overlays
- Redact content with flattened output
- Stamp a handwritten signature onto a PDF
- Lock and unlock PDFs with passwords
- Use the app in English, Espanol, Japanese, or Korean

## Installation note

This prototype is **not notarized yet**. On first launch in macOS:

1. Right-click the app.
2. Click **Open**.
3. Click **Open** again in the confirmation dialog.

## Known limitations

- This release is **Apple Silicon only**
- Intel Mac packaging is not release-ready yet
- Windows packaging is not release-ready yet
- There is no auto-update system yet
